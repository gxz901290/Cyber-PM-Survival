# Cyber PM Survival - AI技术方案

## 1. AI能力概述

本项目集成 **Google Gemini API**，为游戏提供智能化的交互体验。

### 1.1 使用的AI服务
- **Google Gemini API** (via `@google/genai` SDK v1.29.0)
- 模型：Gemini Pro

### 1.2 AI应用场景
- 动态生成职场对话内容
- 智能响应玩家选择
- 个性化事件生成

## 4. AI响应处理

### 4.1 响应解析流程

```typescript
async function generateGameContent(prompt: string): Promise<GameContent> {
  const response = await genAI.models.generateContent({
    model: 'gemini-pro',
    contents: [{ role: 'user', parts: [{ text: prompt }] }],
    systemInstruction: SYSTEM_PROMPT,
  });

  const text = response.text();
  return JSON.parse(text);
}
```

### 4.2 错误处理机制

```typescript
async function safeGenerate<T>(prompt: string, fallback: T): Promise<T> {
  try {
    const result = await generateGameContent(prompt);
    return result;
  } catch (error) {
    console.error('Gemini API Error:', error);
    return fallback;
  }
}
```

## 6. 安全考虑

### 6.1 API密钥保护
- 密钥存储在 `.env.local`，不提交到版本控制
- 通过Vite代理转发API请求，避免前端暴露密钥

### 6.2 内容过滤
- 使用Gemini内置的安全过滤器
- 对生成内容进行二次校验
- 避免生成敏感或不当内容
