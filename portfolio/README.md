# Cyber PM Survival - 项目作品集

## 项目简介

**Cyber PM Survival** 是一款赛博朋克风格的产品经理职场生存模拟游戏。玩家将扮演一名初入职场的PM，在充满"黑话"和"内卷"的科技公司中艰难求生。

## 技术亮点

- **React 19** + **Vite 6** 现代化前端架构
- **Tailwind CSS 4** 原子化CSS方案
- **Motion** 流畅动画体验
- **Gemini API** AI驱动内容生成
- **i18n** 双语国际化支持

## 快速开始

```bash
npm install
npm run dev
```
