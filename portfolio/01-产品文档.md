# Cyber PM Survival - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**Cyber PM Survival** (赛博产品经理生存指南)

### 1.2 产品定位
一款以赛博朋克风格为视觉基调、以互联网产品经理职场生存为核心玩法的文字冒险游戏。玩家扮演一名初入职场的PM，在充满"黑话"和"内卷"的科技公司中艰难求生。

### 1.3 目标用户
- 互联网行业从业者（产品经理、运营、开发等）
- 对职场文化感兴趣的年轻用户
- 喜欢文字冒险/模拟经营类游戏的玩家

---

## 2. 核心玩法

### 2.1 游戏阶段

| 阶段 | 名称 | 描述 |
|------|------|------|
| Phase 1 | 面试 (INTERVIEW) | 玩家回答HR的面试问题，根据回答决定是否入职 |
| Phase 2 | 试用期 (PROBATION) | 新人阶段，完成各种任务，积累属性 |
| Phase 3 | 炼狱期 (PURGATORY) | 中期挑战，面对更复杂的职场困境 |
| Phase 4 | 晋升期 (PROMOTION) | 高级阶段，争取晋升机会 |
| Ending | 结局 | 根据属性和选择，触发不同结局 |

### 2.2 属性系统

| 属性 | 颜色 | 说明 |
|------|------|------|
| Owner意识 | 蓝色 | 对业务的主人翁意识 |
| 沟通能力 | 紫色 | 跨部门协作能力 |
| 逻辑思维 | 金色 | 数据分析与逻辑推理 |
| 站队能力 | 隐藏 | 职场政治敏感度 |
| 理智值 (Sanity) | 红色 | 精神状态，归零则游戏结束 |

### 2.3 结局系统

- **结局 A**: 面试失败，未能入职
- **结局 B**: 试用期被淘汰
- **结局 C**: 晋升失败，继续搬砖
- **结局 D**: 理智归零，精神崩溃

---

## 3. 随机事件系统

### 3.1 飞书弹窗事件
- 每15秒有30%概率触发
- 玩家需快速响应，否则损失理智值和Owner意识

### 3.2 A/B测试轮盘赌
- 每25秒有20%概率触发
- 随机抽取A/B测试结果
- 显著正向：提升逻辑和站队能力
- 负向：损失理智和站队能力

## 4. 视觉设计

### 4.1 配色方案
- 主背景：`#1A1A2E` (深蓝黑)
- 主色调：`#00E5FF` (赛博蓝)
- 强调色：`#E94560` (霓虹粉红)
- 警告色：`#FF003C` (霓虹红)

### 4.2 视觉特效
- CRT显示器扫描线效果
- Matrix数字雨背景
- 霓虹灯光效果 (Neon Glow)
- 毛玻璃面板 (Cyber Glass)

### 4.3 动画效果
- Framer Motion 驱动的流畅过渡
- 打字机效果的文本展示
- 霓虹灯闪烁动画

## 5. 国际化支持

游戏支持双语：
- **English (EN)** - 英文界面
- **简体中文 (ZH)** - 中文界面

用户可在主菜单切换语言。
